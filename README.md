#  Lista de Exercícios: Programação Estruturada

Este repositório contém uma série de exercícios práticos para o treinamento de lógica de programação, cobrindo estruturas de repetição, funções e manipulação de matrizes.

---

##  Estruturas de Repetição

1. **Contagem Crescente**: Imprime os números de 1 a 100 utilizando a estrutura `for`.
2. **Tabuada Dinâmica**: Lê um número inteiro positivo $N$ e exibe a sua tabuada do 1 ao 10.
3. **Análise de 10 Números**: Lê 10 números inteiros e exibe:
   * A soma total dos números.
   * A média aritmética.
   * O maior valor digitado.
4. **Cálculo de Fatorial**: Calcula o fatorial de um número inteiro informado pelo usuário.
5. **Pares com While**: Exibe todos os números pares no intervalo de 1 a 200 utilizando a estrutura `while`.
6. **Acumulador com Parada (0)**: Lê números continuamente até que o usuário digite `0`. Ao final, exibe:
   * A quantidade de números digitados.
   * A soma total.
   * A média dos valores.
7. **Sequência de Fibonacci**: Exibe os primeiros 20 termos da famosa sequência de Fibonacci.

---

##  Funções

* `soma(int a, int b)`: Recebe dois números inteiros e retorna a soma deles.
* `maiorNumero(int a, int b, int c)`: Recebe três números e retorna o maior valor encontrado.
* `ehPar(int n)`: Recebe um inteiro e retorna `1` se o número for par ou `0` se for ímpar.
* `fatorial(int n)`: Função dedicada para calcular e retornar o fatorial de um número.
* `idadeEmDias(int anos)`: Converte a idade de uma pessoa de anos para dias.
* `potencia(int base, int expoente)`: Calcula a potência de forma manual, sem o uso de bibliotecas matemáticas nativas.
* `mediaVetor(int vetor[10])`: Recebe um vetor de 10 posições e calcula a média aritmética dos elementos.

---

##  Matrizes

1. **Leitura 3x3**: Lê uma matriz 3x3 e exibe todos os seus elementos formata2dos na tela.
2. **Análise 4x4**: Lê uma matriz 4x4 e exibe a soma de todos os seus elementos e o maior valor encontrado.
3. **Soma de Matrizes**: Lê duas matrizes 2x2 e gera uma terceira matriz contendo a soma das duas originais.
4. **Diagonal Principal**: Lê uma matriz 3x3 e exibe isoladamente os elementos de sua diagonal principal.
5. **Contador de Pares**: Conta quantos números pares estão presentes em uma matriz de dimensões 5x5.
6. **Verificador de Matriz Identidade**: Lê uma matriz 3x3 e valida se ela corresponde a uma Matriz Identidade.
